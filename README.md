# 📊 CPI Inflation Analysis – India

## 🔍 Overview

This project analyzes India’s Consumer Price Index (CPI) data to uncover key inflation trends and the factors driving price changes over time. The analysis focuses on major categories such as food, energy, and transportation, along with both month-on-month (MoM) and year-on-year (YoY) variations.

## 🎯 Objectives

* Analyze inflation trends across multiple years
* Identify category-wise contribution to overall CPI
* Examine volatility in food inflation
* Assess the impact of external factors like COVID-19 and global oil prices

## 🛠 Tools & Techniques

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization (Charts & Dashboards)

## 📈 Key Insights

* Inflation peaked in **2022**, driven by a surge in global oil prices following the Russia–Ukraine conflict
* Rising fuel and transportation costs significantly impacted overall CPI
* Supply chain disruptions and currency depreciation increased import costs
* Food inflation showed high volatility due to seasonal patterns and climate factors
* Short-term fluctuations (MoM) highlight sharp variations primarily influenced by food prices

## 📊 Analysis Highlights

* Year-on-Year trend analysis to track long-term inflation patterns
* Month-on-Month analysis to capture short-term fluctuations
* Category-level breakdown to understand key contributors to inflation

## 📁 Files Included

* `India_CPI_Inflation.xlsx` – Complete Excel workbook with 10 worksheets
* Dashboard visuals and charts included within the file

## 🚀 Conclusion

The analysis provides a data-driven view of inflation dynamics in India, highlighting how global events, supply disruptions, and sector-specific trends collectively influence price movements.
